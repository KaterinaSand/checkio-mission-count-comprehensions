This is a big module, what should I use first?
First you need to parse the code with ast.parse(source).
You can display the tree with ast.dump. Note that since python 3.9, it has an useful keyword argument "indent". For example: print(ast.dump(tree, indent=4)).
It seems complicated to go through that tree. How should I do it?
There are three ways to go through a tree:
  1. ast.walk: Generates all nodes in the tree in no specified order. Useful if you don't care about the context.
  2. ast.NodeVisitor: Walks the tree and calls a visitor function for every node found (visitor methods you can implement by subclassing this class).
  3. ast.NodeTransformer: Walks the tree and allows modifications.
There are so much classes in this module. What should I use?
You are looking for ast.ListComp, ast.SetComp, ast.DictComp and ast.GeneratorExp.
